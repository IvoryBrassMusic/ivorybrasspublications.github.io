# Nearer, Still Nearer

Instrumentation: String Quartet - two violins, viola, cello  
Duration: 4 minutes, 20 seconds  
Difficulty: Intermediate

---

<b>Description</b>
*Nearer, Still Nearer* is a well-known hymn describing the desire of the believer to fellowship with the Lord. This arrangement of this 1968 hymn uses a string quartet to passionately communicate the soul's desire for God. Suitable for church performance or concert programming, each part of this quartet adds a valuable musical contribution, resulting in an enjoyable listen.
